# java-racingcar-precours

### 기능 정리

- 자동차들 이름 입력받기 ("," 기준 / 영어/한글 상관없이 5자 이하만 가능)
- Message관리 하는 class 생성
- '몇 번의 시도를 할것인가' 입력받기
- 0~9에서 무작위 값 추출 함수 구현
- 각 자동차 함수 넣어서 >=4 이면 전진
- 각자 이름가진 자동차 객체들 생성
- 시도 하나 할 때마다 자동차 위치 출력 함수 구현
- 시도가 목표치에 다다르면 종료.
- 우승한 자동차 이름 출력

### 예외처리

- 사용자가 잘못된 값 입력할 경우 IllegalArgumentException 발생. 어플리케이션 종료
    - 이름 입력시 5자가 넘어갈 경우
        - 이름 입력시 구분자가 "," 가 아닐경우 (당연히 5자가 넘어가겠죠?)
    - 이름이 중복됐을 경우
    - 이름이 공백일 경우
        - ","가 연속으로 두번 나왔을 경우
        - ","로 끝났을 경우
