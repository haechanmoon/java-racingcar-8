# java-racingcar-precourse

## 기능 구현 리스트

- [x] 자동차 이름 입력요구 메시지 출력
- [x] 자동차 이름 입력
- [x] 시도할 횟수 입력요구 메시지 출력
- [x] 시도할 횟수 입력
- [x] 실행결과 메시지 출력
- [x] 각 자동차 객체 분리된 이름으로 생성
- [x] 각 자동차 객체들 랜덤값 실행
- [x] 각 실행 당 자동차 객체들 위치 출력
- [x] 우승자 구하기
- [x] 최종 우승자 메시지 출력

## 예외 처리 리스트 (IllegalArgumentException 처리 후 종료)

- [x] 이름이 5자 초과일때
- [x] 입력이 공백일때 (이름,숫자입력 둘 다)
- [ ] 이름자체가 공백일때 ex)pobi,,woni
- [ ]  시도할 횟수 입력이 1이상의 자연수가 아닐때
- [ ] 이름이 중복됐을 때

## 테스트 리스트

- [ ] 이름이 5자 초과일 때 예외처리 후 종료 확인
- [ ] 입력이 공백일때 예외처리 후 종료 확인
- [ ] 이름이 공백일 때 예외처리 후 종료 확인
- [ ] 시도할 횟수 입력이 [0,1fe,aaa] 일때 예외처리 후 종료 확인
- [ ] 이름이 중복됐을 때 예외처리 후 종료 확인
- [ ] 4이상의 랜덤값을 받았을 때 전진하는지 확인
- [ ] 공동 우승 확인
- [ ] 전진 후 "-"반환 확인
